# 운영 안정성 검증

## 상태

대기 중

## 문제

현재 macOS 운영 구성은 기능 검증과 단기 재시작 검증만 완료됨. 장기간 운영,
절전, 네트워크 단절, 인증서 갱신, 업데이트 실패 상황의 안정성은 아직 확인하지
않음.

## 원하는 동작

MMCP가 장기간 실행과 일반적인 장애 상황에서 자동 복구되며, 안전하게 업데이트
및 백업할 수 있어야 함.

## 범위

### 포함

- 1~2주 장기 실행 중 CPU, 메모리, 저장 공간 사용량 관찰
- macOS 절전, 재부팅, 네트워크 단절 후 자동 복구 검증
- OAuth token refresh와 Let's Encrypt 인증서 자동 갱신 검증
- 업데이트, 재시작, 실패 시 복구 절차 작성
- `.env`와 OAuth signing secret 백업 및 복구 절차 작성

### 제외

- 기능 추가
- Raspberry Pi 이전
