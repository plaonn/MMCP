# 설계 결정

작업 항목이 완료된 뒤에도 이유를 보존해야 하는 장기 결정을 이 디렉터리에
기록함.

다음 조건 중 하나에 해당하면 결정 기록을 생성함.

- 시스템 경계 또는 주요 dependency가 바뀜.
- 보안, 배포, 호환성에 중요한 영향을 줌.
- 미래 작업자가 다시 고려할 만한 현실적인 대안을 제외함.
- 되돌리기 어렵거나 비용이 많이 듦.

`TEMPLATE.md`를 기반으로 작성함. 순번과 짧은 설명으로 파일명을 정함.

```text
0001-use-streamable-http-only.md
```

결정 기록은 역사 자료임. 이후 결정 변경을 숨기려고 이미 승인된 기록을
수정하지 않음. 기존 결정을 대체하는 새 기록을 추가함.
