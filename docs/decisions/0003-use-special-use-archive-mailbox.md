# IMAP 특수 보관 편지함 사용

## 상태

승인됨

## 배경

IMAP 자체에는 이메일 보관이라는 독립 명령이 없음. 보관 기능을 제공하려면
이메일을 적절한 편지함으로 이동해야 하지만, 편지함 이름은 제공자와 사용자
언어 설정에 따라 달라질 수 있음.

네이버 IMAP 계정은 `\Archive` 특수 용도로 표시된 편지함을 하나 제공함.

## 결정

`archive_email`은 서버가 `\Archive` 특수 용도로 표시한 편지함으로 단일
이메일을 이동함.

해당 특수 편지함이 없거나 둘 이상이라 적절한 대상을 결정할 수 없으면 보관
작업을 실패 처리함. 편지함 이름을 추측하거나 임의 설정값으로 대체하지 않음.

## 결과

- 사용자 언어와 편지함 표시 이름에 의존하지 않고 네이버 보관 기능을 제공함.
- 전용 보관 편지함을 제공하지 않는 IMAP 서버에서는 보관 기능을 사용할 수
  없음.
- 보관 작업 후 이메일의 UID가 바뀔 수 있으므로 서버가 새 UID를 제공하면
  결과에 포함함.

## 검토한 대안

- `Archive`, `보관`, `전체메일` 같은 편지함 이름을 추측하는 방안은 언어와
  제공자 설정에 따라 잘못된 편지함을 선택할 수 있어 제외함.
- 환경변수로 보관 편지함을 지정하는 방안은 현재 단일 네이버 계정 목표에
  불필요한 설정과 오작동 가능성을 추가하므로 제외함.
